ФУТБОЛ В МИТИНО — HTML/CSS/JS MVP

1. Положите рядом с index.html файлы:
   players.csv
   stats.csv
   teams.csv
   tournaments_n_rules.csv
   achievements.csv
   hall_of_fame.csv

2. Папку с картинками достижений тоже перенесите без изменения путей из achievements.csv.

3. Сайт нельзя корректно запускать двойным кликом по index.html, потому что браузер блокирует fetch локальных CSV.

Запуск на Windows:
   py -m http.server 8000

Затем открыть:
   http://localhost:8000

Что уже перенесено:
- главная страница;
- выбор турнира и сезона;
- сводка турнира;
- топы;
- статистика по игровым датам;
- личная статистика и диаграмма сравнения;
- достижения и обладатели;
- адаптивная верстка.

Следующий этап:
- точный перенос Bayesian Overall и расчета предпочтительных партнеров;
- сортировка таблиц кликом;
- красивые URL и отдельные страницы при необходимости.
